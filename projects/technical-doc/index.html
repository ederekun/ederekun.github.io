Debian Installation Guide

Installation instructions for the Debian GNU/Linux distribution. The manual describes the installation process using the Debian Installer, the installation system for Debian that was first released with Sarge (Debian GNU/Linux 3.1).
Additional information related to the installation can be found in the Debian Installer FAQ and the Debian Installer wiki pages.

Authors:

Debian Installer team

Maintainer:

Debian Installer team

Status:

The manual is not yet perfect. Active work is being done for the current and future Debian releases. Help is welcome, especially with non-x86 installation and translations. Contact debian-boot@lists.debian.org for more information.

Availability:

Debian package installation-guide

Published version for the stable release
Available on the official CDs and DVDs in the /doc/manual/ directory.

Version being prepared for the next stable (in testing)

Development version

The latest XML source is available through the Git repository.

  • Web interface: Link to online installation guide
  • VCS interface: git clone https://salsa.debian.org/installer-team/installation-guide.git

Versions of the installation guide for previous releases (and possibly the next release) of Debian are linked from the release page for those releases.

Securing Debian Manual

This manual describes the security of the Debian GNU/Linux operating system and within the Debian project. It starts with the process of securing and hardening the default Debian GNU/Linux installation (both manually and automatically), covers some of the common tasks involved in setting up a secure user and network environment, gives information on the security tools available, steps to take before and after a compromise and also describes how security is enforced in Debian by the security team. The document includes a step by step hardening guide and within the appendix there is detailed information on how to setup an intrusion detection system and a bridge firewall with Debian GNU/Linux.

Authors:

Alexander Reelsen, Javier Fernández-Sanguino Peña

Maintainer:

Javier Fernández-Sanguino Peña

Status:

Published; under development with few changes. Some of its content might not be fully up-to-date.

Availability:

Debian package harden-doc

Latest version is available in:
English, German, Spanish, French, Italian, Japanese, Portuguese, and Chinese

The latest XML source is available through the Git repository
